# Campaign API

## Overview

The Campaign API allows you to create and manage Campaigns and Insertion Orders (IOs). This page covers the common endpoints and methods associated with the Campaign API.

  * **Campaigns** are targeted advertising runs that operate for a set period of time under specified optimization strategies. 
  * **Insertion Orders** are the parent hierarchy to Campaigns and they define the contracts between an Advertiser and a media partner, as well as the parameters of an advertising Campaign.

### Get List of Campaigns

<span class="badge badge--primary">GET</span> <span class="path-text">/api/v2/cmp/campaigns/data</span>

<div class="container">
  <div class="child1">

Returns list of Campaigns based on set of filters and <var>conversionId</var> query parameter.

A Campaign <var>id</var> can be used to [Get Campaign Details by ID](#get-campaign-details-by-id) or to perform various actions in [Campaign Management](#campaign-management).

| Query Parameters |  |
| ---- | --- |
| `searchField` <br /><span class="type-text">string</span> | Search results by keyword |
| `limit` <br /><span class="type-text">integer</span> | Maximum number of entries returned, default: <var>50</var> |
| `pageNo` <br /><span class="type-text">integer</span> | Page number for the data, default: <var>1</var> |
| `sortBy` <br /><span class="type-text">string</span> | Sorts by ascending (<var>+</var>) or descending (<var>-</var>), default: <var>-created</var> | 
| `conversionId` <br /><span class="type-text">integer</span> | Targeted Campaign will be returned at top of list |
| `status` <br /><span class="type-text">string</span> | Filter by multiple statuses |
| `owIds` <br /><span class="type-text">integer</span> | Filter by Organization Workspace IDs |
